# TxtPhoto

- 40KB以下までのtxtを画像に変換します。
- [Image2Ascii](https://github.com/coara-chocomaru/Image2Ascii)で生成したDATを画像に復元します。
- 生成した画像の保存場所は/sdcard/picture/txtphoto/です。

## 概要
小さめのtxtをサクッと画像に変換するアプリです。

※個人的に使ってるだけのアプリです🙇
